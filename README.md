# TaskFlow — Mobile (teste da turma)

Protótipo **mobile** do TaskFlow para a turma testar a experiência de captura rápida de tarefas.

- **Organizar** é a tela inicial: caixa "Nova tarefa" no topo (Enter salva) + chips **Hoje / Amanhã / Sem data**.
- Botão **＋** (em qualquer aba) abre a captura rápida (tarefa ou nota) e **fica aberto** para lançar várias em série.
- Abas: **Organizar · Atrasadas · Hoje · Amanhã · Solicitadas · Notas**.
- **Solicitadas** tem sub-abas *Solicitadas* (o que você pediu a outros) e *Confirmações* (o que outros concluíram e aguardam seu OK).

## Observações
- É **protótipo de UX**. Os dados são salvos **no próprio navegador de cada pessoa** (localStorage) — **não** são compartilhados entre usuários (isso é papel do backend real).
- Para limpar seus dados de teste: use a função de reset (ou limpe os dados do site no navegador).
- Identidade do usuário é **simulada** (Diego), só para demonstrar as regras.
